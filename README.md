## Tech stack

- Android 5.0 (API level 21) or later
- Kotlin
- Coroutine + Flow : Asynchronous event handling
- Dagger-Hilt : Dependency injection
- Jetpack
    - LiveData
    - ViewModel
    - Lifecycle
    - Room
    - Navigation
    - Startup
